# Git Show & Diffs

## git show

* show comit and contents

```text
git show <commit>
```

* show files changed in commit

```text
git show <commit> --stat
```

* look at a file from another commit

```text
git show <commit>:<file>
```

## git diff

* unstaged changes

```text
git diff
```

* staged changes

```text
git diff --staged
```
